# Test whether an object is a ReticulateMuData

Test whether an object is a ReticulateMuData

## Usage

``` r
is_reticulate_mudata(x)
```

## Arguments

- x:

  Any R object.

## Value

`TRUE` if `x` inherits from `"ReticulateMuData"`, `FALSE` otherwise.

## Examples

``` r
is_reticulate_mudata(42)        # FALSE
#> [1] FALSE
```
