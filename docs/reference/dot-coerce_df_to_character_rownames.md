# Ensure a data.frame has character row names

Ensure a data.frame has character row names

## Usage

``` r
.coerce_df_to_character_rownames(df, default_prefix = "obs")
```

## Arguments

- df:

  A data.frame.

- default_prefix:

  Prefix used when row names are missing.

## Value

The same data.frame with valid character row names.
